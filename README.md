# APIOK Admin

APIOK 网关管理后台系统

## 技术栈

- Vue 3
- Ant Design Vue
- Vue Router
- Vuex
- Axios
- NProgress

## 功能模块

- 服务管理
- 路由管理
- 上游管理
- 证书管理
- 插件管理

## 开发

```bash
npm install
npm run serve
```

## 构建

```bash
npm run build
```

## 环境配置

开发环境需要配置 `VUE_APP_ROOT` 环境变量，指向后端 API 地址。

## 浏览器支持

- Chrome
- Firefox
- Safari
- Edge

不支持 IE 11
